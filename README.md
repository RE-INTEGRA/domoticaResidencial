# Domótica Residencial

Este projeto visa monitorar a rede elétrica de uma casa possibilitando certo controle sobre  a mesma sem necessariamente ter que mexer manualmente ou estar presente.


##Funcionalidades 
 
1- Controlar remotamente via wifi o sistema de iluminação para ligar e/ou desligar lampadas de acordo com a necessidade ou gosto.
2- Controlar os eletrodomésticos remotamente via wifi para que se possa ter um controle maior e mais possibilidades gerando mais conforto, por exemplo por meio de um aplicativo poder ligar o ar-condicionado da casa 10 minutos antes de vc chegar em casa para que a casa já esteja com uma tempatura mais amena.
3- Ativar automaticamente os portões da casa por meio de sensores que detectam quando o carro está se aproximando, ou então ser ativado remotamente via wifi.
4- Desenvolver um aplicativo para mobile para que todas as funcionalidades anteriores  possam ser controladas por ele.
